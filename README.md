# Gesturely AI - Sign Language Translator

## Overview
Gesturely AI is a real-time sign language translation system that converts hand gestures into text and text into sign language representations. The project uses Computer Vision and Deep Learning to improve communication accessibility for people with hearing and speech impairments.

## Features
- Real-time sign language recognition
- Sign-to-text conversion
- Text-to-sign conversion
- Interactive web interface
- Deep learning-based gesture classification

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- MediaPipe
- Flask
- HTML, CSS, JavaScript

## Installation

### Clone Repository

```bash
git clone https://github.com/nafeesaps/Gesturely-AI-sign-language-translator.git
```

### Move to Project Directory

```bash
cd Gesturely-AI-sign-language-translator
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python main.py
```
